# OnBoardingRite

Backend API and Frontend application for the OnBoarding project.

## Project Structure
- `backend/` - FastAPI backend application
- `src/` - React frontend application
- `schema_ddl.sql` - Database schema

## Tech Stack
- **Backend**: FastAPI, SQLAlchemy, PostgreSQL
- **Frontend**: React, TypeScript, Vite

## API Endpoints
- Dashboard API
- Projects API
- Checklists API
- Requisitions API
- Eligibility Rules API
